Class: Scope

Numbas.jme. Scope

new Scope(scopes)

JME evaluation environment.

Parameters:
Name Type Description
scopes Numbas.jme.Scope[]

List of scopes to combine into this one. Scopes appearing later in the list override ones appearing earlier, in case variable/ruleset names conflict.
Properties:
Name Type Description
variables object

dictionary of Numbas.jme.token objects
functions object

dictionary of arrays of Numbas.jme.funcObj objects. There can be more than one function for each name because of signature overloading.
rulesets objects

dictionary of Numbas.jme.Ruleset objects
